Codex Provider Sync

切换登录方式时同步会话历史

这是什么工具？

Codex CLI 支持两种登录方式：OpenAI 官方 OAuth 和 第三方 API Key。 两种方式的会话历史是隔离的。本工具可以在你切换登录方式前，把历史会话迁移到目标 provider， 让你在新登录方式下也能看到之前的所有会话。

使用方式：切换登录前，先选择目标 provider，点击"同步历史"。 例如：从 API 登录切到官方登录前，选择 openai 并同步。
